# Data Sources and Download Instructions

All data used in this study are publicly available. Due to licensing restrictions
on redistribution, raw data files must be downloaded from the original sources.

## 1. Market Prices — Agmarknet

**Source**: Agricultural Marketing Information Network  
**URL**: https://agmarknet.gov.in  
**What to download**: Daily wholesale modal prices for wheat and rice  

### Manual download steps:
1. Go to https://agmarknet.gov.in → "Daily Report" → "Prices and Arrivals"
2. Select commodity: "Wheat" or "Rice"
3. Select state and market from the list below
4. Set date range: 01-Jan-2020 to 31-Dec-2024
5. Download as CSV/Excel
6. Save to `data/raw/agmarknet/`

### Markets used (10 APMCs across 5 states):
| State | Markets |
|-------|---------|
| Punjab | Amritsar, Ludhiana |
| Haryana | Karnal, Hisar |
| Uttar Pradesh | Lucknow, Kanpur |
| Madhya Pradesh | Indore, Bhopal |
| West Bengal | Kolkata, Burdwan |

**Note**: Agmarknet sometimes requires CAPTCHA verification. The scraping script 
(`src/01_data_collection.py`) provides a semi-automated approach but may need 
manual intervention. Alternatively, use the bulk download option when available.

Expected file naming: `{state}_{market}_{commodity}_{year}.csv`

## 2. Meteorological Data — India Meteorological Department

**Source**: India Meteorological Department  
**URL**: https://www.imd.gov.in/pages/services_hydromet.php  
**Alternative**: https://power.larc.nasa.gov/data-access-viewer/ (NASA POWER, for backup)

### Download steps:
1. Go to IMD data portal or use `mausam.imd.gov.in`
2. Select "District-wise Rainfall" and "Max/Min Temperature"
3. Select relevant states and date range (2020–2024)
4. Save to `data/raw/imd/`

Variables needed:
- Daily rainfall (mm)
- Maximum temperature (°C)
- Minimum temperature (°C)
- Relative humidity (%)

The preprocessing script derives additional variables:
- 7-day cumulative rainfall
- Temperature anomaly (deviation from 30-day moving average)
- Extreme heat days (days where max temp > 40°C)
- Drought indicator (rainfall below 25th percentile of monthly distribution)

## 3. Procurement and Policy Data — FCI & CACP

**FCI procurement volumes**: Available from FCI Annual Reports at https://fci.gov.in  
**MSP announcements**: CACP reports at https://cacp.dacnet.nic.in  

These are compiled in `data/msp_procurement.csv` (included in this repository)
since they are small and publicly cited in government press releases.

## 4. International Reference Prices

**FAO**: https://www.fao.org/faostat → Trade → Prices  
**World Bank**: https://www.worldbank.org/en/research/commodity-markets  

Download monthly wheat and rice international reference prices.
Save to `data/raw/international/`
